## HTML
`HTML (Hypertext Markup Language)` adalah bahasa markup standar yang digunakan untuk membuat struktur dan `kerangka dasar halaman website.`

HTML menggunakan kode-kode berbentuk tag—seperti <head>, <p>, dan <img>—untuk mengatur konten seperti teks, gambar, dan hyperlink agar dapat ditampilkan oleh web browser.

### Berikut adalah poin-poin penting mengenai HTML:
- Fungsi Utama: Membangun struktur kerangka halaman web (header, footer, konten, link).
- Komponen Dasar: Terdiri dari tag (pembuka/penutup), element (isi di dalam tag), dan attribute (informasi tambahan).
- Bukan Bahasa Pemrograman: HTML adalah bahasa markup (penandaan) yang statis, bukan bahasa pemrograman logis.
- Versi Terbaru: Saat ini yang digunakan secara umum adalah HTML5 (dirilis 2014) yang lebih dinamis dan mendukung multimedia.
- Pondasi Web: HTML bekerja bersama CSS (tampilan/styling) dan JavaScript (interaktivitas) untuk menciptakan situs web modern.

### Contoh Sederhana

    <!DOCTYPE html>
    <html>
    <head>
        <title>Halaman Saya</title>
    </head>
    <body>
        <h1>Selamat Datang</h1>
        <p>Ini adalah paragraf pertama saya.</p>
    </body>
    </html>

### Taq Umum Pada HTML

1. Tag Struktur Utama:
- `<html>` : Tag pembungkus seluruh elemen HTML pada halaman web.
- `<head>` : Berisi informasi meta tentang dokumen (tidak tampil di halaman web), seperti judul tab browser dan link ke CSS.
- `<body>` : Membungkus seluruh konten yang akan terlihat oleh pengunjung di halaman web.

2. Tag Semantik (Pembagian Area Web)
- `<header>` : Mendefinisikan bagian kepala web, biasanya berisi judul situs atau logo.
- `<nav>` : Wadah khusus untuk kumpulan link navigasi (menu).
- `<main>` : Menandakan area konten utama dari dokumen tersebut.
- `<footer>` : Menandakan bagian kaki web, biasanya berisi copyright atau informasi kontak.

3. Tag Teks dan Konten
- `<h1>` sampai `<h6>` : Digunakan untuk membuat heading (judul atau subjudul). `<h1>` adalah yang paling besar dan penting.
- `<p>` : Mendefinisikan sebuah paragraf teks.
- `<strong>` : Membuat teks menjadi tebal (bold) sekaligus memberi penekanan makna.
- `<em>` : Membuat teks menjadi miring (italic) untuk memberi penekanan.
- `<small>` : Membuat teks berukuran lebih kecil, biasanya untuk keterangan tambahan seperti tanggal.
- `<br>` : (Break line) Membuat garis baru atau enter ke bawah.
- `<hr>` : (Horizontal rule) Membuat garis lurus membentang untuk memisahkan konten.

4. Tag Tautan dan Daftar
- `<a>` : (Anchor) Membuat tautan/hyperlink untuk berpindah ke halaman lain.
- `<ul>` : (Unordered List) Membuat daftar item yang tidak berurutan (biasanya berupa titik/bullet).
- `<li>` : (List Item) Mendefinisikan setiap item di dalam daftar <ul>.

5. Tag Pengelompokan
- `<div>` : (Division) Wadah serbaguna untuk mengelompokkan elemen-elemen lain agar mudah diatur tampilannya dengan CSS.

### Atribute pada Taq HTML
Atribut memberikan informasi tambahan atau pengaturan khusus pada sebuah tag. Atribut selalu ditulis di dalam tag pembuka.
