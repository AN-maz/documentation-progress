# Pertemuan 1 

## Apa itu Web Developer?
profesi seseorang yang bertugas untuk menjadi pembuat dan pengembang website (world wide web). 
Tidak hanya sebagai pembuat, web developer juga memastikan kinerja website tetap berjalan stabil dan baik. 

Biasanya `web developer` juga sering disebut sebagai `programmer`. 
Sebab, profesi ini mengharuskan untuk memahami cara membuat website dari nol serta maintenance agar kinerja website tetap stabil dan optimal.  

Web developer sendiri dibagi menjadi tiga jenis berdasarkan peran dan scope pekerjaan yang dikuasai, diantaranya:

1. `Back-end developer` adalah programmer website yang berperan untuk memastikan website selalu optimal, cepat, aman dan berfungsi dengan baik.  

2 `Front-end developer` adalah seseorang yang bertugas untuk mendesain tampilan website supaya lebih menarik, dengan penunjang fitur-fitur yang dibutuhkan.  

3 `Full-stack developer` adalah orang yang melakukan dua jobdesk sekaligus yaitu, pekerjaan front end dan juga back end. Sehingga, untuk menjadi full stack developer dibutuhkan skill yang mampu memaksimalkan tampilan website begitu juga dengan fitur-fitur, serta menjaga keamanan dari sisi server.  

## Tugas Web Developer
Tugas web developer sebenarnya dibedakan menurut jenis, peran dan scopenya seperti yang sudah dijelaskan sebelumnya yakni, front end, back end atau full stack developer.  

berikut beberapa tugas utama web developer yang perlu diketahui:

1. `Pembuat dan Pengembang Website`
Peran web developer: Back -end developer

Pertama, tugas web developer adalah menjadi pembuat serta pengembang website. Web developer bertanggung jawab agar website lebih responsif, mudah diakses, digunakan dengan menggunakan satu atau lebih bahasa pemrograman.

Selain website, web developer juga biasanya bertugas untuk mengoptimalkan aplikasi yang berbasis web.

2. `Membuat dan Mengetes API`
Peran: Back end developer

Berikutnya tugas web developer dalam menjalankan perannya sebagai back end developer adalah untuk membuat dan mengetes API (Application Programming Interfaces). API sendiri adalah komponen penting untuk pengembangan website untuk menghubungkan situs dengan komponen software.
 

3. `Membuat Database`
Peran: Back-end developer

Sebagai back end developer, tugas web developer satu ini juga bertugas membuat database. Database ini yang akan digunakan untuk menyimpan, gambar, video, akun pengguna dan lain sebagainya.
 

4. `Mendesain Tampilan (Interface) dan Navigasi Web`
Peran: Front-end developer

Tugas web developer selanjutnya adalah mendesain interface dan navigasi web. Biasanya hal ini dilakukan oleh web developer yang berperan sebagai front end developer. Tujuan dari tugas web developer untuk mendesain interface agar pengunjung tidak bosan ketika mengakses web, serta merancang fitur-fitur dengan navigasi yang efisien, sehingga website dapat mudah diakses dan digunakan.
 

5. `Maintenance dan Mengamankan Website`
Peran: Back-end developer

Banyaknya serangan hacker dan cyber crime, maka tugas web developer adalah maintenance serta mengamankan website. Lebih lanjut, web developer perlu mencari celah keamanan website pada server serta memperbaiki keamanan website. Dengan begitu, website akan selalu aman dengan serangan hacker dan kepercayaan pengunjung website. 

## Skill yang Dibutuhkan Web Developer

1. `HTML/CSS`: Keterampilan dasar dalam merancang dan memformat struktur halaman web menggunakan HTML (Hypertext Markup Language) dan CSS (Cascading Style Sheets). 

2. `JavaScript`: Bahasa pemrograman yang digunakan untuk memberikan interaktivitas pada halaman web. JavaScript juga sering digunakan untuk mengelola logika klien (client-side) pada aplikasi web. 

3. `Framework dan Library`: Pengetahuan tentang framework seperti React, Angular, atau Vue.js dapat membantu membangun aplikasi web yang kompleks secara lebih efisien. Selain itu, familiaritas dengan library seperti jQuery juga dapat bermanfaat. 

4. `Backend Development`: Pengembangan server dan logika di sisi server menggunakan bahasa pemrograman seperti Node.js (JavaScript), Python (Django, Flask), Ruby (Ruby on Rails), PHP, atau Java. 

5. `Database Management`: Pemahaman tentang sistem basis data seperti MySQL, PostgreSQL, MongoDB, atau Firebase untuk menyimpan dan mengelola data aplikasi web. 

6. `HTTP/REST`: Pengetahuan tentang protokol HTTP dan konsep RESTful API untuk berkomunikasi antara frontend dan backend. 

7. `Version Control/Git`: Kemampuan untuk menggunakan sistem kontrol versi seperti Git untuk mengelola dan menyinkronkan kode bersama tim pengembangan.

8. `Responsive Design`: Mampu membuat desain yang responsif, yang berarti dapat beradaptasi dengan baik pada berbagai perangkat dan ukuran layar. 

9. `Testing/Debugging:` Keterampilan dalam menguji dan melakukan debugging pada aplikasi web untuk memastikan kinerja yang baik dan menemukan dan memperbaiki bug. 

10 `Keamanan Web`: Memahami praktik keamanan web, termasuk perlindungan terhadap serangan umum seperti Cross-Site Scripting (XSS) dan SQL injection. 

11. `Pengalaman Pengguna (UX):` Memahami prinsip-prinsip desain antarmuka pengguna untuk menciptakan pengalaman pengguna yang baik.

12. `Task Runners dan Build Tools`: Menggunakan alat seperti Webpack, Gulp, atau Grunt untuk otomatisasi tugas-tugas pengembangan, pengujian, dan penyebaran kode.

13. `Pemahaman Basis Data`: Pengetahuan tentang dasar-dasar basis data dan cara berinteraksi dengan mereka melalui bahasa query. 

14. `Pemahaman Cloud Computing`: Pengetahuan tentang platform cloud seperti AWS, Azure, atau Google Cloud Platform dapat bermanfaat untuk menyebarluaskan aplikasi dan mengelola sumber daya. 

15. `Keahlian Khusus`: Misalnya, jika Anda bekerja dalam pengembangan game web, mungkin diperlukan keahlian tambahan dalam WebGL atau pengembangan game.

## Apa itu website 
Website adalah `sekumpulan halaman di internet yang saling terhubung dan bisa dibuka melalui browser` seperti Chrome atau Firefox.

Di dalam website, kita bisa menemukan `berbagai hal seperti tulisan, gambar, video, sampai fitur interaktif`. Website biasanya digunakan untuk `berbagi informasi, belajar, berbisnis, atau hiburan`.

### Komponen Utama Website:
- Domain : Alamat website (contoh: google.com)
- Hosting :Tempat menyimpan semua data website agar bisa diakses online
- Web Browser : Aplikasi untuk membuka website (Chrome, Firefox, dll)
- Homepage :  Halaman utama saat pertama kali membuka website

### Fungsi Website:
- Sumber informasi : Menyediakan berita, artikel, atau materi pembelajaran
- Media bisnis :  Digunakan untuk jual beli produk atau jasa secara online
- Personal branding : Menampilkan portofolio atau blog pribadi
- Sarana komunikasi : Memudahkan interaksi dengan orang lain (chat, kontak, dll)

### Jenis Website:
- Website statis :  Isinya jarang berubah
- Website dinamis : Isinya bisa sering diperbarui dan lebih interaktif
