Menjelajahi Dunia Web Development

Ditulis oleh Purwa pada 19 Maret 2026

Memahami HTML dan CSS adalah langkah awal yang krusial bagi setiap pengembang web. Walaupun sekarang sudah banyak alat canggih, menguasai dasarnya membuat kita lebih paham bagaimana web sebenarnya bekerja.


Taq semantik seperti <header>, <main>, dan <Footer> tidak hanya membuat kode lebih rapih, tapi juga bagus untuk SEO dan aksesibilitas.

Selanjutnya, sy berencana untuk terus berlatih dengan membuat layout multi-page dan mengeksplorasi properti CSS lainnya!
