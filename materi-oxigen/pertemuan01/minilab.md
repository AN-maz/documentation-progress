## BAGIAN 2: MINI LAB (60%)

**Studi Kasus:** Web Portofolio Multi-Page Sederhana

Di sesi ini, teman-teman akan membuat kerangka awal web portofolio pribadi yang terdiri dari 2 halaman dan bisa saling terhubung.

### Misi 1: Setup Struktur Folder & HTML (Kerangka Utama)
1. Buat folder baru bernama `portfolio-ku`.
2. Di dalamnya, buat dua file HTML: `index.html` (untuk Beranda) dan `about.html` (untuk Profil Singkat).
3. Di `index.html`, buat struktur HTML5 dan tambahkan navigasi (`<nav>`) yang berisi link ke `index.html` dan `about.html`.
4. Tambahkan elemen ucapan selamat datang (`<h1>`) dan deskripsi singkat (`<p>`).
5. Lakukan hal yang sama di `about.html`, tapi isi kontennya dengan profil singkat diri kalian, hobi, atau *skill*.
6. *Test:* Buka di *browser* dan pastikan kalian bisa bolak-balik antar halaman dengan mengklik link di navigasi!

### Misi 2: Styling Dasar dengan CSS
1. Buat file `style.css` di folder yang sama.
2. Hubungkan `style.css` ke dalam `<head>` di file `index.html` dan `about.html` menggunakan tag `<link>`.
3. Atur warna latar belakang (`background-color`), jenis huruf (`font-family`), dan ubah warna teks agar lebih menarik.
4. Beri `padding` pada `<main>` agar teks tidak menempel ke pinggir layar.

### Misi 3: AI Challenge (Eksplorasi Navigasi & Tampilan)
Sekarang, gunakan bantuan AI untuk merapikan tampilan portofolio kalian!
1. Buka AI (Gemini/ChatGPT).
2. Tulis *prompt* untuk meminta AI membuatkan **CSS Navbar (Menu Navigasi)** yang menarik (misalnya: menu berjejer ke samping, ada warna latar, dan warnanya berubah saat di-*hover*).
3. Tulis *prompt* lain untuk meminta AI membuatkan **tampilan Card sederhana** menggunakan CSS untuk membungkus profil kalian di halaman `about.html`.
4. Terapkan kodenya dan pelajari bagian mana dari CSS tersebut yang mengubah tampilannya!
