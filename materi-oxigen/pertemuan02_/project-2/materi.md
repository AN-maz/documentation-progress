## Pendahuluan 
Pada pertemuan sebelumnya, kita telah mengenal dasar-dasar CSS sebagai bahasa styling untuk mempercantik tampilan website.
Pada pertemuan ini, pembahasan akan lebih mendalam pada bagaimana CSS mengatur ukuran, jarak, posisi, dan susunan elemen agar halaman website terlihat lebih rapi, modern, dan responsif.

Materi ini penting karena dalam pengembangan website modern, seorang developer tidak hanya harus bisa memberi warna atau font, tetapi juga harus mampu:

- Menyusun layout halaman dengan rapi
- Membuat tampilan website fleksibel di berbagai ukuran layar
- Menulis kode CSS yang bersih dan mudah dirawat
- Menghindari penulisan kode berulang yang tidak efisien

Setelah mempelajari materi ini, diharapkan mahasiswa mampu memahami konsep box model, menggunakan Flexbox dan CSS Grid, membuat desain responsive, serta melakukan code refactoring pada stylesheet.
