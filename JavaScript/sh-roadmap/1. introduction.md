# Apa itu Java Script? 
JavaScript (JS) adalah bahasa pemrograman tingkat tinggi yang digunakan untuk membuat situs web menjadi interaktif dan dinamis. Bersama HTML dan CSS, JavaScript merupakan teknologi inti dalam pengembangan web.

# Sejarah Java Script
Diciptakan oleh `Brendan Eich` pada `tahun 1995` saat ia bekerja di Netscape Communications.

- Pengembangan Cepat: Prototipe awalnya diselesaikan hanya dalam waktu 10 hari.
- Perubahan Nama: Awalnya dinamakan Mocha, kemudian LiveScript, dan akhirnya menjadi JavaScript sebagai taktik pemasaran agar populer bersama bahasa Java yang saat itu sedang naik daun.
- Standarisasi: Pada tahun 1997, JavaScript diserahkan ke ECMA International untuk distandarisasi dengan nama resmi ECMAScript.

# Versi JavaScript (ECMAScript)
Standar resmi JavaScript disebut ECMAScript (ES). Sejak 2016, versi baru dirilis setiap tahun menggunakan penamaan tahun:
- Versi Klasik: ES1, ES2, ES3, ES5, dan ES6 (ECMAScript 2015) yang membawa perubahan besar dalam sintaks.
- Versi Tahunan: ES2016 hingga yang terbaru ES2024 (ES15).
- Masa Depan: Saat ini sedang dalam tahap pengembangan untuk ES2025 (ES16) yang akan menyertakan metode baru seperti pemrosesan Set yang lebih canggih.

# Cara Menjalankan JavaScript
JavaScript tidak memerlukan instalasi khusus karena sudah didukung oleh semua peramban (browser) web modern.

1. Di Browser (Console):
    1. Buka browser (Chrome/Firefox/Edge).
    2. Tekan tombol F12 atau klik kanan > Inspect > tab Console
    3. Ketik kode seperti `console.log("Hello World");` dan tekan Enter.

2. Melalui File HTML:
    1. Tulis kode di dalam tag `<script>` dalam file HTML
    2. Atau hubungkan file eksternal dengan format .js menggunakan atribut src.

3. Di Server (Node.js):
    1. Instal Node.js di komputer Anda.
    2. Buka terminal atau Command Prompt.
    3. Ketik perintah `node nama_file.js` untuk menjalankan skrip tersebut di luar browser.
