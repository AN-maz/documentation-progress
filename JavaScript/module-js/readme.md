# Module materi JavaScript

Module ini dirancang sesuai dengan alur pembelajaran fundamental *web development*, dipersiapkan sebagai fondasi utama sebelum melangkah ke ekosistem yang lebih kompleks seperti React.js atau pengembangan *backend* dengan Node.js. 

Modul ini terbagi menjadi beberapa bagian yang terstruktur, sangat cocok digunakan sebagai bahan panduan dalam sesi pengajaran, kegiatan *Tech Talk*, maupun untuk praktik belajar mandiri. Setiap bagian dilengkapi dengan penjelasan teori, contoh kode, serta latihan (studi kasus) interaktif.

## Daftar Modul Pembelajaran

### 1. Pengenalan dan Dasar-Dasar JavaScript
Bagian ini mencakup fondasi awal bahasa pemrograman JavaScript.

- `Variabel & Tipe Data`: Penggunaan `let`, `const`, serta pemahaman tipe data primitif (`String`, `Number`, `Boolean`).
- `Operator & Logika`: Operator aritmatika, penugasan, logika (`AND`, `OR`, `NOT`), dan struktur percabangan `if/else`.
- `Perulangan (Loops)`: Implementasi `while` dan `for` loop untuk iterasi data.
- `Fungsi (Functions)`: Pembuatan fungsi dasar, parameter, argumen, dan pengembalian nilai (return).
- `Struktur Data`: Pengelolaan sekumpulan nilai menggunakan Array (termasuk metode `forEach`) dan Object.

### 2. Pengenalan Document Object Model (DOM)
Bagian ini berfokus pada interaksi antara JavaScript dan antarmuka HTML untuk menciptakan halaman web yang dinamis.
- `DOM Selection`: Teknik menyeleksi elemen menggunakan `getElementById`, `querySelector`, dan `querySelectorAll`.
- `DOM Manipulation`: Mengubah teks (`textContent`, `innerHTML`), atribut, dan modifikasi inline style.
- `CSS Class Manipulation`: Menambah, menghapus, dan toggle class menggunakan `classList`.
- `Node Manipulation`: Membuat elemen baru (`createElement`) dan menghapusnya dari struktur DOM.
- `DOM Events`: Penanganan interaksi pengguna (klik, input, submit) menggunakan `addEventListener` dan `pemanfaatan Event Object`.
- `Studi Kasus`: Pembuatan fitur Dark Mode, Counter, dan To-Do List interaktif.
