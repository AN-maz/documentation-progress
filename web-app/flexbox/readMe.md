# Tabel Properti Flexbox

| Properti        | Kegunaan                              | Nilai Populer              |
|-----------------|--------------------------------------|-----------------------------|
| flex-direction  | Arah sumbu utama                      | row, column                 |
| justify-content | Perataan Horizontal (Main Axis)      | center, space-between, end  |
| align-items     | Perataan Vertikal (Cross Axis)        | center, stretch, flex-start |
| gap             | Jarak antar elemen                    | 10px, 1rem                  |
| flex-grow       | Kemampuan elemen untuk melebar        | 0 (tetap), 1 (tumbuh)        |
